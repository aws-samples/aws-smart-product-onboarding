# Smart Product Onboarding API Handlers

This package contains the Lambda function implementations for the Smart Product Onboarding API.

## Overview

This package provides:

- Lambda function handlers for API endpoints
- Business logic implementation
- Integration with other workspace packages

## Structure

The handlers will be moved here from `packages/api/handlers/python/` as part of the workspace migration.

## Dependencies

- `amzn-smart-product-onboarding-api-runtime` - Generated API runtime (workspace dependency)
- AWS SDK and related tools for Lambda development

## Usage

These handlers are deployed as AWS Lambda functions and handle:

- Product data generation requests
- Product categorization requests
- Attribute extraction requests
- Status and health check endpoints
